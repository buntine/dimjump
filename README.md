## Warning

This game may cause extreme frustration and/or feelings of inescapable anxiety.

You've been warned.

## Instructions

 - *UP*: Jump
 - *DOWN*: Duck
 - *LEFT*: Go slower
 - *RIGHT*: Go faster
 - *P*: Pause
 - *S*: Toggle sound

## Running

To start a web server for the application, run each of these commands in a separate tab:

    lein ring server-headless
    lein figwheel
